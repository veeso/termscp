# 打开与编辑文件

termscp 可以使用外部应用程序打开文件，并在你配置的编辑器中编辑文本文件。本地和远程文件均受支持。

## 打开与使用指定程序打开

按 `<V>` 可使用系统针对该文件类型的默认应用程序打开文件。termscp 依赖于你操作系统的默认打开程序（由 [open](https://docs.rs/crate/open/1.7.0) crate 提供支持），因此请确保你的系统上至少具备以下之一。

- **Windows**：通过 `start` 命令自动处理。
- **macOS**：通过 `open` 自动处理，该命令已预装。
- **Linux**：必须安装 `xdg-open`、`gio`、`gnome-open` 或 `kde-open` 之一。
- **WSL**：需要 `wslview`。请安装 [wslu](https://github.com/wslutilities/wslu)。

按 `<W>` 可使用你指定的程序打开文件。

## 编辑文本文件

按 `<O>` 可在你配置的文本编辑器中打开文件。仅支持文本文件，不支持二进制文件。

如果文件位于远程主机上，它会先被下载到你的临时文件目录，然后**仅在**你对其进行了更改时才会重新上传到远程主机。termscp 通过检查文件的最后修改时间来检测更改。

## 编辑远程文件

你无法直接从远程面板就地编辑远程文件。当你打开远程文件时，它会被下载到临时目录，但 termscp 无法创建监视器来检测你用于打开它的外部程序何时关闭，因此无法判断你何时完成编辑。要编辑远程文件，请先将其下载到本地目录，在本地编辑后再重新上传。
