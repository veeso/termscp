# 保持文件同步

文件监视器会将本地路径同步到远程路径。当在被监视的本地路径上检测到更改时，该更改会在大约 5 秒内传播到配置的远程路径。

你可以监视任意数量的路径。

## 监视路径

1. 将光标置于本地浏览器上你想保持同步的文件或目录上。
2. 在远程面板中，导航到应报告更改的目录。
3. 按 `<T>`。
4. 在弹窗中回答 `<YES>`。

## 取消监视路径

你可以通过两种方式停止监视某个路径：

- 在被监视的本地路径（或其任意子文件夹）上按 `<T>`。
- 按 `<CTRL+T>`，然后在你想取消监视的路径上按 `<ENTER>`。

## 传播的更改

以下更改会被报告到远程主机：

- 新建文件和文件更改
- 文件被移动或重命名
- 文件被删除或取消链接

监视器仅在单一方向上工作（从本地到远程）。在远程主机上所做的更改**不会**被同步回本地主机。
